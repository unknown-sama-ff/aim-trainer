Aim Trainer — FPS 瞄准训练系统

一、概述

Aim Trainer 是一款基于浏览器的 FPS 瞄准训练系统，采用纯前端单文件架构，无需安装任何依赖即可运行。系统以 Canvas 2D 为渲染核心，搭配固定时间步长游戏循环（144Hz 逻辑帧率），确保训练过程的流畅性与精确性。该应用面向 FPS 游戏玩家，提供多种瞄准维度的专项训练模式，辅以详尽的统计数据与评级体系，帮助玩家系统性地提升鼠标操控能力与射击反应速度。

二、训练模式

1. 网格射击（Grid Shot）
网格射击是系统的核心训练模式。屏幕被划分为 40×40 的细密网格，目标以绿色圆形呈现，同屏最多可同时出现 12 个目标。该模式采用"聚集生成"机制——同一批次的目标会在相邻网格区域内集中出现，模拟实战中敌人从同一方向涌来的场景，训练玩家的快速多目标切换能力。击中目标后，新目标会在聚集区内补位，同时系统会记录近期被击中的格子并短暂屏蔽该区域，避免目标重复出现在同一位置。玩家可自定义目标尺寸、同屏数量、总目标数、出现间隔和聚集半径等参数。
2. 追踪训练（Tracking）
追踪训练模式要求玩家持续将准星对准一个沿预设轨迹移动的目标球体。系统提供五种运动路径：水平、垂直、圆形、8 字形和随机，速度可在 100 至 500 px/s 之间调节。训练结束后，系统根据追踪覆盖率（准星在目标上的时间占比）和平均偏差距离计算得分。该模式默认启用 360° 全景视角，玩家通过鼠标转动视角来追踪目标，更贴近 FPS 游戏中的跟枪体验。训练时长支持 15 秒、30 秒和 60 秒三档。
3. 反应测试（Reaction）
反应测试模式专注于测量玩家的视觉反应速度。目标在随机延迟后出现，玩家需尽快点击。若在目标出现前提前点击，系统会判定为"抢跑"并记录失误。目标设有超时限制（默认 2 秒），超时未命中同样计为失误。训练结束后，系统会剔除 ±2σ 范围外的异常值后计算平均反应时间，确保统计结果的可靠性。测试次数可选 10、25 或 50 次。
4. 扫射转移（Flick Shot）
扫射转移模式模拟 FPS 中常见的"跟枪"操作。球体沿选定轨迹移动，玩家需要按住鼠标左键持续跟踪球体，保持准星在目标上达到指定时长（默认 3 秒）即算成功。跟踪进度以环形进度条实时显示，松开鼠标或准星偏离目标则进度中断。球体移动轨迹同样支持水平、垂直、圆形、8 字形和随机五种路径，速度和跟踪时长均可自定义。

三、360° 全景模式

所有四种训练模式均支持 360° 全景视角。启用后，鼠标从直接控制准星位置变为控制相机偏航角与俯仰角，准星锁定在屏幕中心，目标以球面坐标投影到屏幕上。场景背景包含星空、城市天际线剪影、透视网格线和方位标识（N/E/S/W），营造沉浸式的空间感。当目标移出视野时，屏幕边缘会出现绿色箭头指示器，引导玩家转向目标方向。该模式通过 Pointer Lock API 锁定鼠标，实现与 FPS 游戏一致的视角操控体验。

四、自定义设置

系统提供丰富的个性化设置选项。灵敏度支持 0.1 至 10 倍调节，可开启原始输入模式以绕过操作系统鼠标加速。准星支持三种样式：十字、圆点和圆环，颜色、粗细、长度、中心间隙和透明度均可自由调节，还可开启射击扩散动画模拟后坐力效果。此外，音效和屏幕震动效果可独立开关。所有设置通过 localStorage 持久化保存，刷新页面后自动恢复。

五、统计与评级

每次训练结束后，系统展示详细的结算面板，包括得分、准确度、平均点击时间、最大连击等核心指标，并根据综合表现给出 S/A/B/C/D 五级评定。评级算法综合考量准确度权重（40%）、反应速度、追踪覆盖率和得分等多个维度。系统自动保存最近 50 条训练记录和各模式最高分，支持历史趋势图表查看，便于玩家追踪长期进步。
